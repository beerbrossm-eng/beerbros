🔗 เชื่อม Google Sheet
ทำตามขั้นตอนนี้ครั้งเดียว แล้วใช้ได้ 2 คนพร้อมกันตลอด
1
สร้าง Google Sheet ใหม่
ไปที่ sheets.new แล้วตั้งชื่อว่า "BEER BROS" จากนั้นสร้าง Sheet 4 แผ่นชื่อ:
Products, Sales, Expenses, Restocks
2
ติดตั้ง Apps Script
ใน Google Sheet → Extensions → Apps Script → ลบโค้ดเดิม → วางโค้ดนี้แล้วกด Save (Ctrl+S)
// คลิกเพื่อ copy โค้ด Apps Script
3
Deploy เป็น Web App
ใน Apps Script → Deploy → New deployment → Execute as: Me → Who has access: Anyone → Deploy → Copy URL
4
